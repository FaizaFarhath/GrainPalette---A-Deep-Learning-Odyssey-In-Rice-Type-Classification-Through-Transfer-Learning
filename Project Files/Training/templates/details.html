GrainPalette – Rice Type Classification

Project Overview

This project uses deep learning to automatically classify different types of rice grains based on their image. The web interface allows users to upload an image of a rice grain, and the system predicts the rice type using a pre-trained CNN model.

Objective

  • To build a deep learning model that can classify rice grain types accurately.
  • To create a user-friendly web interface using Flask.
  • To support farmers and researchers in quickly identifying rice types.

Technologies Used

  • Python & Flask (Web Framework)
  • TensorFlow & Keras (Deep Learning)
  • HTML/CSS (Frontend)
  • VS Code (IDE)

Model Architecture

The model is built using transfer learning with MobileNetV2 pre-trained on ImageNet. It has been fine-tuned on a custom rice grain dataset with multiple rice classes (e.g., Basmati, SonaMasoori, Jasmine, etc.).

Dataset

The dataset consists of thousands of labeled rice grain images categorized into different classes. It is split into training, validation, and testing sets.

How It Works

  • User uploads an image via the homepage.
  • The image is processed and fed into the trained model.
  • The predicted rice type and confidence level are displayed.
